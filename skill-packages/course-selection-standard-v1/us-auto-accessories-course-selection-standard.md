# 美国大单品配件：汽车配件课程筛选标准 v1

## 目标

用于筛选“美国市场 + 大单品配件 + 汽车配件”相关课程。重点服务前期选品决策，而不是泛泛学习跨境运营。

最终要学会判断：

- 哪类汽车配件适合小白切入
- 哪些数据能证明需求真实
- 哪些产品因为合规、适配、售后风险必须避开
- 如何把课程内容拆给策略、数据、运营、视觉四个能手

## 汽车配件方向的课程筛选模型

总分 100。

### 1. 汽配业务匹配度 30 分

高分课程必须直接回答：

- 汽车配件/汽车后市场/车载用品怎么选
- 美国 Amazon 汽配类目怎么看
- 哪些车系/车型/场景有配件机会
- 哪些汽配产品风险高，不能碰

扣分：

- 只讲泛亚马逊选品
- 只讲开店和运营，不讲选品判断
- 只讲国内汽配，不涉及跨境和美国用户

### 2. 前期选品决策价值 25 分

高分课程必须有：

- 选市场而不是只选产品
- 关键词、竞品、评论、价格、利润、供应链的判断路径
- 类目门槛和风险判断
- 可复制的选品表或评分逻辑

扣分：

- 只给爆品清单
- 只讲个人感觉
- 只说“这个类目好卖”

### 3. 数据和工具实操 15 分

高分课程包含：

- Amazon 搜索、Best Sellers、BSR、评论、Q&A
- ABA / Brand Analytics / Product Opportunity Explorer / 卖家精灵 / Keepa / Helium 10 / Jungle Scout 等工具方法
- 供应端价格和美国端价格对比

扣分：

- 没有数据来源
- 把估算销量当真实销量
- 不讲如何验证

### 4. 风险意识 10 分

汽配风险比普通配件高。高分课程必须提到：

- 车型适配风险
- 安全责任
- 认证/法规
- 商标和车型词使用
- 退货和售后
- 大件/易损/安装复杂

低分或淘汰：

- 鼓励做刹车、灯具、电池、电子控制、排放、承重、安全件，但不讲风险
- 教侵权跟卖或规避平台规则

### 5. 时效性 10 分

规则、类目、广告、工具、平台页面优先 2024-2026。

老课只保留底层方法：

- 市场选择逻辑
- 评论痛点分析
- 竞品拆解方法

### 6. 可信度和可转化性 10 分

高分课程：

- 作者长期讲跨境/亚马逊/汽配
- 有真实案例或工具演示
- 能转成四个能手任务单

低分课程：

- 招商引流重
- 标题党
- 承诺暴富
- 公开视频没干货

## 课程等级

- A 级：80-100，必须学，学完拆成技能包
- B 级：65-79，提取模块
- C 级：50-64，只看观点
- D 级：50 以下，跳过

## 一票否决

出现以下情况直接淘汰：

- 教刷单、刷评、侵权、规避风控
- 不讲汽配合规和适配风险
- 只卖课、招商、私信领取，公开内容无实操
- 明显过时且没有底层价值
- 要求泄露账号、密码、后台、密钥

## 眼前优先学哪些课

### P0 必学：汽配选品决策

1. 汽车配件跨境选品逻辑
2. 美国汽车后市场/汽车配件类目认知
3. Amazon 汽配类目竞品分析
4. 车型适配和兼容表达
5. 汽配合规和高风险产品避坑

### P1 必学：Amazon 数据选品

1. Amazon 市场分析和产品分析
2. ABA/关键词/搜索需求分析
3. Product Opportunity Explorer 使用方法
4. 评论痛点和 Q&A 分析
5. 利润、体积重量、退货风险评估

### P2 补充：运营和视觉

1. 汽配 Listing 标题和兼容表达
2. 主图、尺寸图、安装图、适配图
3. 广告关键词和转化复盘
4. 供应商筛选、打样、质检
