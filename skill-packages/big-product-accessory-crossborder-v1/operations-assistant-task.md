# 给运营助手的任务单

## 目标

把策略助手选出的配件方向变成可上架、可展示、可测试、可复盘的执行方案。

## Listing 必备模块

标题：

- 产品类型
- 适配对象
- 核心功能
- 数量/套装
- 关键材质或场景

五点描述：

1. 解决什么痛点
2. 兼容哪些型号
3. 材质/质量/安全说明
4. 使用方法和场景
5. 套装内容、售后和注意事项

主图/副图：

- 主图：清楚展示产品和套装内容
- 图 2：使用前后对比
- 图 3：适配型号表
- 图 4：尺寸图
- 图 5：安装/使用步骤
- 图 6：材质/细节/质量
- 图 7：场景图或礼品化场景

## 短视频脚本模板

### 痛点型

- 0-2 秒：展示主品使用麻烦
- 3-6 秒：展示配件解决动作
- 7-12 秒：展示前后对比
- 13-18 秒：展示适配和细节
- 19-25 秒：展示套装和购买理由

### 对比型

- 普通配件的问题
- 我们的改良点
- 使用效果对比
- 适合哪些人

### 安装型

- 开箱
- 安装步骤
- 注意事项
- 使用完成效果

## FAQ 模板

必须覆盖：

- 是否兼容某型号
- 尺寸是多少
- 套装包含什么
- 是否官方原装
- 如何安装
- 如何清洁
- 不适配怎么办

注意：涉及品牌主品时，用“compatible with”表达兼容，不暗示官方授权。

## 差评预防清单

上架前必须检查：

- 适配型号是否明确
- 尺寸图是否清楚
- 包装是否防压防刮
- 说明书是否能避免误用
- 标题和图文是否过度承诺
- 是否有容易误解的品牌表达
