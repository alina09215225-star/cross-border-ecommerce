# B站跨境学习路线

## 学习目标

服务“大单品配件”主线，不追求泛学跨境。先建立亚马逊选品和运营基础，再补 TikTok 内容测试、独立站和合规。

## 第一批课程

### 1. 新手现实感

- 课程：废热跨境日记《纯新手小白如何选跨境平台和选品》
- 链接：https://www.bilibili.com/video/BV15M4y1j7hS
- 学习目的：理解新手从 0 进入时的平台选择、选品误区和现实约束
- 输出：3 个新手避坑点，3 个适合小白的验证动作

### 2. 跨境选品方法

- 课程：白小菌-外贸跨境《四种跨境电商选品方法,如何寻找Trending Products》
- 链接：https://www.bilibili.com/video/BV1QVxve3EzM
- 学习目的：掌握广告选品、平台选品、工具选品、英文信息源选品
- 输出：大单品配件候选池采集方法

### 3. 亚马逊 FBA 选品逻辑

- 课程：佛系跨境小僧《吃透亚马逊FBA选品逻辑》
- 链接：https://www.bilibili.com/video/BV17o4y1T7va
- 学习目的：建立 Amazon 选品和竞争判断框架
- 注意：课程较早，规则和工具细节需用最新官方信息校准
- 输出：亚马逊配件机会评分表

### 4. 亚马逊平台全流程

- 课程：长空频道《亚马逊FBA运营系列教程，零基础亚马逊跨境电商运营教程合集》
- 链接：https://www.bilibili.com/video/BV1uiQ6YAEA5
- 学习目的：理解注册、上架、FBA、基础运营、Listing、广告等全流程
- 输出：运营助手执行地图

### 5. 亚马逊运营和广告

- 课程：佛系跨境小僧《吃透亚马逊运营+广告逻辑》
- 链接：https://www.bilibili.com/video/BV1XMvseTETD
- 学习目的：理解上架后如何获得曝光、转化和复盘
- 输出：配件产品启动和广告验证框架

## 学习顺序

1. 先看新手现实感，避免被大卖打法带偏
2. 再看选品方法，建立候选池
3. 再看亚马逊选品逻辑，形成评分模型
4. 再看亚马逊平台全流程，知道产品怎么落地
5. 最后看运营和广告，解决怎么卖出去
