# 参考来源

本技能包基于以下公开资料和方法框架整理：

- Amazon Product Opportunity Explorer 官方页面：https://sell.amazon.com/tools/product-opportunity-explorer
- Amazon Seller University / Learn 官方学习入口：https://sell.amazon.com/learn
- Amazon 商品 Listing 和详情页优化相关官方资料：https://sell.amazon.com/blog/amazon-product-listings
- Amazon Brand Registry / 品牌保护相关官方资料：https://sell.amazon.com/brand-registry

使用要求：

- 涉及费用、规则、政策、认证、广告后台、Brand Registry 时，必须以最新官方页面为准。
- BSR、评论数、第三方工具估算只能作为代理指标，不能写成真实销量。
- 商标、专利、认证风险只能做初筛，最终要专业确认。
