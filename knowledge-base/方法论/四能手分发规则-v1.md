# 四能手分发规则 v1

## 总原则

任何学习内容都不只做总结，必须拆成四类可执行能力。

## 策略（选品）能手

接收内容：

- 市场机会
- 类目逻辑
- 车型/人群/场景选择
- 产品筛选标准
- 进入和放弃标准
- SKU 组合策略
- 阶段化路线

输出形式：

```markdown
# 策略技能包
## 做什么
## 为什么做
## 适合谁
## 评分模型
## 进入标准
## 放弃标准
## 给其他能手的任务
```

## 数据能手

接收内容：

- 数据源
- 关键词
- 趋势验证
- 竞品采样
- 评论/Q&A 分析
- 利润测算
- 运营复盘指标

输出形式：

```markdown
# 数据技能包
## 数据源
## 字段表
## 采样方法
## 判断规则
## 报告模板
## 给其他能手的结论
```

## 运营能手

接收内容：

- 上品流程
- Listing 结构
- 发货履约
- 库存补货
- 促销
- 客服
- 退货和下架

输出形式：

```markdown
# 运营技能包
## 上品前检查
## Listing 模板
## 发货库存表
## 促销规则
## 售后分类
## 日报/周报模板
```

## 视觉能手

接收内容：

- 主图
- 副图
- 尺寸图
- 适配图
- 安装图
- A+
- 视频脚本
- 广告素材

输出形式：

```markdown
# 视觉技能包
## 图片结构
## A+ 模板
## 视频脚本
## 类目视觉重点
## 禁止项
## 素材清单
```

## 分发命名规则

```text
主题-v版本-角色.md
```

示例：

```text
us-auto-accessories-amazon-v2-strategy-selection.md
us-auto-accessories-amazon-v2-data-analysis.md
us-auto-accessories-amazon-v2-operations-fulfillment.md
us-auto-accessories-amazon-v2-visual-creative.md
```
