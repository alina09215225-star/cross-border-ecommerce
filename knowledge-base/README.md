# 跨境学习知识库

## 目的

这里是跨境学习能手的总知识库。以后所有学习内容先沉淀到这里，再按角色拆分给四个能手。

## 固定流程

1. 找资料：B站、平台官方文档、卖家案例、工具报告、竞品页面、评论和论坛。
2. 筛价值：只保留能影响选品、数据、运营、视觉执行的内容。
3. 做总库：沉淀课程摘要、核心方法论、可执行 SOP、风险清单。
4. 分角色：分别写入策略、数据、运营、视觉四个能手文件夹。
5. 留记录：在 `分发记录` 里记录资料来源、输出文件和下一步动作。

## 文件结构

```text
知识库/
  README.md
  课程学习/
    每门课程或资料的结构化笔记
  方法论/
    可反复复用的选品、数据、运营、视觉方法
  分发记录/
    每次学习后分发给四个能手的记录
```

## 输出标准

每次学习必须沉淀以下内容：

- 资料来源和可信度
- 核心观点
- 系统性方法论
- 可执行 SOP
- 风险和限制
- 对四个能手的具体任务
- 可复用表格或模板
