# Cross-Border Ecommerce Knowledge Base

这个仓库用于同步跨境电商相关的 Codex 工作资料、技能包、研究流程和交接文档。目标是让另一台电脑 clone 后，可以直接继续学习、选品、数据分析、运营和视觉创意工作。

## 目录结构

- `agents/learning/`: 学习能手，主要负责课程学习、视频/字幕转学习材料。
- `agents/strategy-selection/`: 策略（选品）能手，放置后续沉淀的选品判断、平台策略和品类路线。
- `agents/data-analysis/`: 数据能手，主要负责市场数据、评论、竞品和评分分析。
- `agents/operations/`: 运营能手，放置刊登、履约、客服、库存和流程 SOP。
- `agents/visual-creative/`: 视觉能手，放置主图、详情页、短视频、广告素材方法和模板。
- `skill-packages/`: 已沉淀的完整方向包，按平台/品类/版本保存。
- `shared/`: 跨能手共用的术语、评分框架、研究模板和报告模板。
- `archives/`: 原始压缩包归档，用于必要时还原。

## 已迁移内容

- `bilibili-course-learning`: B 站课程学习与材料整理流程。
- `global-ecommerce-product-intelligence`: 全球电商选品、评论和市场研究流程。
- `amazon-accessory-product-research-v1`: Amazon 配件产品研究方向包。
- `big-product-accessory-crossborder-v1`: 大单品配件跨境方向包。
- `shopee-taiwan-scooter-accessories-v1`: Shopee 台湾机车配件方向包。
- `us-auto-accessories-amazon-v1`: 美国汽车配件 Amazon 方向包。
- `course-selection-standard-v1`: 课程筛选标准。

## 使用方式

1. 新线程或新电脑先读 `HANDOFF.md`。
2. 根据任务类型进入对应能手目录。
3. 如果任务是完整品类/平台研究，优先读 `skill-packages/` 下对应方向包。
4. 新产出的稳定方法、模板、SOP 放回 `shared/` 或对应能手目录。
