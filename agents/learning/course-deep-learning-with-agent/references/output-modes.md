# 输出模式

## 1. 20% 先行模式

适合用户时间少，先抓主线。

输出：

- 课程核心问题
- 一句话主判断
- 3 个关键原因
- 1 个主要边界
- 2 个追问建议

## 2. 五层完整模式

适合真正要学透。

输出：

- 结论层
- 原理层
- 证据层
- 边界层
- 迁移层
- 检验层

## 3. 追问式教学模式

适合用户想边学边问。

节奏：

1. 先讲核心问题和主判断
2. 停下等待用户追问
3. 再展开证据和边界
4. 再迁移到业务场景

## 4. 检验式教学模式

适合用户想确认自己是不是真的理解了。

输出：

- 5 个判断题 / 追问题
- 用户容易答错的地方
- 标准答案和原因
- 如果答错，回到哪一层重新解释
