# 学习能手

职责：课程筛选、课程学习、视频/字幕处理、知识点提取、行动计划生成。

当前资料：

- `bilibili-course-learning/`: B 站课程学习技能包。
- `course-deep-learning-with-agent/`: 课程深度学习技能包，用于把课程总结升级为结论、原理、证据、边界、迁移、检验六层结构。
- `../../skill-packages/course-selection-standard-v1/`: 课程筛选标准。

常用入口：

1. 先用课程筛选标准判断课程是否值得学。
2. 对值得学的课程，用 `bilibili-course-learning` 获取字幕、文稿和基础学习材料。
3. 对课程总结、课程复盘、课程迁移应用，默认套用 `course-deep-learning-with-agent` 的六层深度学习结构。
4. 把学习结果转成可执行清单，交给策略、数据、运营或视觉能手继续。

## 默认课程总结标准

以后所有“学过课程后的总结”，优先按下面结构输出：

1. 这门课到底在回答什么问题
2. 一句话主判断
3. 为什么会得出这个判断
4. 证据链
5. 边界与反例
6. 常见误解 vs 正确理解
7. 如果放到我的场景里
8. 5 个检验问题
9. 最后再给行动方案

目标不是把课程压缩得更短，而是让用户真正学懂、会讲、会判断、会迁移。
