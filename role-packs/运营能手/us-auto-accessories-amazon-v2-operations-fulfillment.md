# 运营能手：美国汽配上品、履约、售后 v2

## 来源

- 课程：《跨境汽配选品逻辑》BV1T1421r7ps
- 本技能包提炼课程中关于 SKU 覆盖、易出单、低售后、A+、促销、下架的运营方法。

## 运营总原则

汽配运营不是只上一个爆款，而是围绕“SKU 组合”持续做加减法。

核心目标：

- 新店先出单。
- 页面讲清适配，降低退货。
- 用促销拉动高客单产品，但不能倒贴。
- 对高售后 SKU 及时下架。

## 上品前检查

每个 SKU 上架前确认：

```csv
sku,product_type,vehicle_fitment,year_range,install_position,dimensions,package_content,safety_risk,legal_risk,return_risk,listing_ready,images_ready,a_plus_ready
```

必须清楚：

- 适配车型、年份、位置。
- 不适配情况。
- 安装是否需要工具。
- 是否影响行车安全。
- 是否使用了未授权车标、品牌名、OEM 表述。

## 新店启动 SKU

优先上：

- 容易出单。
- 售后低。
- 适配简单。
- 客单价不低到促销后亏损。
- 可批量复制同类目/同车型/同场景。

暂缓上：

- 需要专业安装的核心件。
- 安全责任高的刹车、承重、灯光、电子控制件。
- 退货主要来自不适配的产品。
- 只能靠低价亏损出单的产品。

## Listing 规则

标题结构：

```text
[Product Type] Compatible with [Vehicle/Year], [Install Position], [Core Benefit], [Material/Pack]
```

五点结构：

1. 适配范围：车型、年份、位置。
2. 核心功能：解决什么使用问题。
3. 安装方式：是否需要工具，安装步骤。
4. 材质和套装：尺寸、数量、包装。
5. 不适配和售后：明确限制，减少误买。

禁止：

- official、genuine、OEM 等未授权表述。
- 夸大安全、防撞、防盗、防水。
- 隐藏不适配条件。

## 促销和优惠券

课程重点：促销适合高客单/有利润空间的 SKU，低价品容易倒贴。

执行前测算：

```csv
sku,sale_price,coupon_value,coupon_cost,platform_fee,shipping_cost,return_allowance,net_profit,net_margin,promo_decision
```

规则：

- 低客单 SKU 不做重券。
- 高客单 SKU 可用优惠券拉动成交，但要保留退货损耗。
- 促销目标必须明确：开单、清库存、冲排名、测需求。

## A+ 和批量页面

课程提到 A+ 图文页能提升转化，但逐个精做不现实。

运营动作：

- 先按类目建模板：控制臂、雨挡、收纳、车斗配件等。
- 只对有订单/高曝光/高客单 SKU 优先做精细 A+。
- 批量上传前筛掉不支持 A+ 的 SKU。
- A+ 必须包含适配、安装、尺寸、材质、FAQ。

## 售后分类

每日记录：

```csv
date,sku,asin,order_id,issue_type,vehicle_model,customer_message,return_reason,action,next_owner
```

问题类型：

- 不适配
- 尺寸误解
- 安装困难
- 质量/破损
- 包装损坏
- 页面误导
- 物流延迟

反馈规则：

- 不适配：给视觉能手补适配图，给策略能手复核是否继续。
- 安装困难：补安装图/视频/说明书。
- 质量问题：暂停补货，查供应端。
- 高退货低利润：建议下架。

## 下架规则

满足任意两条，提交策略能手决定下架：

- 30 天无订单且无有效曝光。
- 有订单但退货率高。
- 差评集中在不可解决的适配问题。
- 促销后仍亏损。
- 客服成本明显高于利润。

## 每日运营模板

```markdown
# 美国汽配运营日报

## 今日订单
SKU、订单、异常。

## 上品动作
新增、修订、待补资料。

## 促销动作
优惠券、价格、盈亏。

## 页面动作
标题、五点、A+、图片、视频。

## 售后
退货、差评、Q&A、客服问题。

## 加减法
建议加码 SKU；建议下架 SKU。

## 需要协作
策略决策；数据验证；视觉修改。
```
