# 运营能手：Shopee 台湾机车配件上品、发货、日常运营 v1

## 角色定位

负责 Shopee 台湾机车配件的上架、标题、规格、价格、物流、客服、库存、发货、退货和日常复盘。

## 上品前检查

必须确认：

- 商品是否影响行车安全
- 是否涉及认证、法规、侵权或夸张安全承诺
- 尺寸和适配车型是否清楚
- 材质、颜色、规格是否和实物一致
- 是否需要繁体中文说明
- 包装是否能防压、防刮、防水
- 是否适合店到店/宅配

## Shopee 标题结构

```text
機車[配件類型] [核心功能] [適用場景/車型] [材質/尺寸] [數量/套組] 現貨
```

示例结构：

- `機車手機架 防震防滑 後照鏡款 導航支架 外送通勤適用`
- `機車杯架 飲料架 免打孔 可調式 外送通勤 機車百貨`

## 商品页必须写清楚

- 适用车型/安装位置
- 不适用情况
- 尺寸
- 材质
- 安装方式
- 套装内容
- 出货地
- 到货时效
- 售后处理

## 规格/变体

优先用变体提高转化：

- 颜色
- 数量
- 安装方式
- 尺寸
- 套装组合

避免过多复杂变体导致发错货。

## 发货和库存表

```csv
date,product,sku,variation,available_inventory,daily_orders,reorder_point,supplier_status,packing_status,shipping_method,tracking,delayed_orders,return_count,notes
```

## 客服 FAQ

必须准备繁体中文：

- 這款適合我的車嗎？
- 可以裝在 Gogoro / Yamaha / Kymco 嗎？
- 會不會鬆動？
- 下雨可以用嗎？
- 怎麼安裝？
- 尺寸是多少？
- 什麼時候出貨？
- 不合適可以退換嗎？

## 日常运营检查

每日：

- 是否有新订单未出货
- 是否有库存风险
- 是否有买家聊聊未回复
- 是否有差评或退货
- 广告是否异常
- 热销变体是否缺货

每周：

- 调整标题关键词
- 优化主图和规格图
- 汇总客服问题给数据能手
- 汇总退货原因给策略能手
