# Scoring Framework

## 基础评分维度

- 需求强度：是否刚需，是否有明确搜索意图。
- 品牌依赖：非头部品牌是否有机会成交。
- 竞争强度：头部卖家垄断程度、价格战程度、广告成本压力。
- 供应稳定性：是否容易找货、是否规格稳定、是否易质检。
- 履约难度：重量、体积、破损率、退货率、售后复杂度。
- 合规风险：安全、认证、平台禁限售、侵权风险。
- 内容可做性：图片、视频、详情页是否容易做出差异。
- 兼职适配度：是否适合有限时间运营，是否需要高频客服或复杂定制。

## 推荐结论

- A：优先验证，可进入样品/小批量测试。
- B：可以观察或小规模验证。
- C：暂不优先，除非有供应链或流量优势。
- D：放弃。
